## 基于vite插件和vitest测试工具实现一下需求

1. 开发一个vite插件，手动控制vue组件更新渲染，自定义组件更新的时机和更新逻辑

2. 不使用 vue 自带的 $forceupdate 来强制更新组件

3. 不影响原始组件渲染逻辑

4. 不改变原始组件DOM结果

5. 有的组件会存在没有根节点包裹的情况出现

6. 当前vite插件只需要支持vue3就可以，

7. 组件有类型有 .vue组件，.tsx组件，h函数渲染的组件都要支持

8. 使用typescript 开发插件

9. 插件需要支持数组格式配置匹配，匹配到的组件进行改写注入逻辑

10. 使用vitest编写测试用例，覆盖全面、边界测试清晰，可以提供完整的测试覆盖率
